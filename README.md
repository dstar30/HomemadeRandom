# Random Variate Generator
This package provides access to a few frequently used discrete and continuous distributions.


## Installation
### Normal Installation
`pip install HomemadeRandom`

## Distributions

### Discrete Distributions
1. Bernoulli
2. Binomial
3. Geometric
4. Poisson

### Continuous Distributions
1. Uniform
2. Exponential
3. Normal
4. Gamma
5. Weibull
6. Triangular
